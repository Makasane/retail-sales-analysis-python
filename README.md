# Retail Sales Analysis (Python)

## Project Overview
This project analyzes a simulated retail sales dataset to uncover:
- Revenue trends
- Category performance
- Customer purchasing behavior

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
This dataset is simulated and created for educational and portfolio purposes.

## Key Analysis Performed
- Data cleaning and inspection
- Monthly revenue analysis
- Revenue by product category
- Top 10 customer analysis
- Data visualization using charts

## Key Insights
- Revenue trends can be tracked monthly
- Certain categories contribute more to total revenue
- A small group of customers generate a large portion of sales

This project demonstrates foundational data analysis skills using Python.
